## Overview
This project is a pixel-perfect implementation of the **JTG Frontend Home Assignment**, developed strictly as per the provided guidelines and Figma mock-ups.

The webpage is built using **HTML5, CSS3, and JavaScript only**, without using any CSS or JavaScript frameworks or UI libraries.

---

## Technologies Used
- HTML5 (semantic elements)
- CSS3
- JavaScript

---

## Features
- Pixel-perfect UI matching the Figma design
- Fully implemented sections as per the mock-up
- Working carousel/slider
- Modal implementation without external libraries
- Cross-browser compatible (Chrome, Firefox, Safari)
- Clean, optimized, and maintainable code
- Responsive layout for desktop viewports

---

## Guidelines Followed
- No CSS/JS frameworks or UI libraries used
- No tables for layout
- Semantic HTML5 tags used
- Open Sans font integrated from Google Fonts
- Assets exported only from Figma
- Best coding and optimization practices followed
